条件付き期待値の計算¶

簡単な線形モデル \begin{eqnarray} y = x + v\ x～N(\overline{x},\sigma^2_a)\ v～N(0,\sigma^2_v) \end{eqnarray}

正規分布の加法性¶

\begin{eqnarray} a ～ N(\mu_a, \sigma_a^2)\\ b ～ N(\mu_b, \sigma_b^2)\\ であるとき\\ a + b も正規分布に従い、\\ a + b ～ N(\mu_a+\mu_b, \sigma_a^2+\sigma_b^2) となる。 \end{eqnarray}

事後確率密度と条件付き期待値¶

ベイズの定理¶

\begin{eqnarray} p(x_i \mid y_i) &=& \frac {集合「x=xi」における、y=yiとなる事象} {集合「全x」における、y=yiとなる事象全て} \\&=& \frac {p(y_i \mid x_i)p(x_i)} {\int p(y_i \mid x)p(x)dx} \end{eqnarray}

事後確率密度関数は \begin{eqnarray} p(x \mid y) = \frac {1} {\sqrt{2 \pi \tau^2}}

\exp(¶

\frac{1}{2 \tau^2}
(x - \alpha \tau^2)^2

) \end{eqnarray}

ここで \begin{eqnarray} \frac{1}{\tau^2} = \frac{1}{\sigma_a^2} + \frac{1}{\sigma_v^2} \ \alpha = \frac{\overline{x}}{\sigma_a^2} + \frac{y}{\sigma_v^2} \end{eqnarray}

よって条件付き期待値は \begin{eqnarray} \alpha \tau ^2 = \frac{1}{\sigma_a^2+\sigma_v^2} (\sigma_v^2 \overline{x} + \sigma_a^2 y) \end{eqnarray}

yの条件付き期待値とxの条件付き期待値¶

\begin{eqnarray} y = x + v,　x～N(\overline{x},\sigma^2_a),　v～N(0,\sigma^2_v) \end{eqnarray}

における \begin{eqnarray} && p(y \mid x)　と　p(x \mid y) \end{eqnarray}

yの条件付き期待値¶

\begin{eqnarray} p(y \mid x) \end{eqnarray}

については、xが確定されたなら、 \begin{eqnarray} y = x + v \end{eqnarray} より、期待値がxで、分散がvの分散と等しい確率分布となることが分かる。よって \begin{eqnarray} p(y \mid x) = \frac {1} {\sqrt{ 2 \pi \sigma_v^2 }} \exp(- \frac { (y - x)^2 } {2 \sigma_v^2} ) \end{eqnarray}

xの条件付き期待値¶

では \begin{eqnarray} p(x \mid y) \end{eqnarray} について同じ論法が使えるか？つまり、

yが確定されたなら、 \begin{eqnarray} x = y - v \end{eqnarray} より、期待値がyで、分散がvの分散と等しい確率分布と考えられる?

結論から言えばNO。

1.
　まず、「x：原因」「y：結果」という違いがある。

2.
　x=x1が確定した後にyを求めるケースでは、
原因：x1 + ランダムノイズ　→ 結果：y より
「yの平均がx1であることは自明」である。
　一方、y=y1が確定した後にxを求めるケースでは、
y1が生成された過程は未知であり、
y=y1を満たすxとvの組み合わせは様々考えられる。
よって、ただちに「xの平均がy1であることは自明」と言うことはできない。

3.

yが確定した x = y - v
