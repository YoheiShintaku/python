事後確率密度と条件付き期待値¶

ベイズの定理¶

\begin{eqnarray} p(x_i \mid y_i) &=& \frac {x=xiy=yi} {xy=yi} \\&=& \frac {p(y_i \mid x_i)p(x_i)} {\int p(y_i \mid x)p(x)dx} \end{eqnarray}

\begin{eqnarray} p(x \mid y) = \frac {1} {\sqrt{2 \pi \tau^2}}

\exp(¶

\frac{1}{2 \tau^2}
(x - \alpha \tau^2)^2

) \end{eqnarray}

\begin{eqnarray} \frac{1}{\tau^2} = \frac{1}{\sigma_a^2} + \frac{1}{\sigma_v^2} \ \alpha = \frac{\overline{x}}{\sigma_a^2} + \frac{y}{\sigma_v^2} \end{eqnarray}

\begin{eqnarray} \alpha \tau ^2 = \frac{1}{\sigma_a^2+\sigma_v^2} (\sigma_v^2 \overline{x} + \sigma_a^2 y) \end{eqnarray}
